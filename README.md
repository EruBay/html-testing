# 🧪 html-testing

HTML/CSS/JS testing playground.

## Structure

- `index.html` — Landing page
- Add new experiments as separate `.html` files

## Local Dev

Just open any `.html` file in your browser, or use a local server:

```bash
# Python
python3 -m http.server 8080

# Node
npx serve .
```
